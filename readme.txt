Floating WhatsApp chat widget for WordPress. Customizable button, inquiry options, and basic analytics.

== Description ==
A customizable floating WhatsApp chat widget for WordPress websites. Easily add a chat button, set inquiry options, and track interactions. Powered by flowfunnel.io.

== Frequently Asked Questions ==
= Can I customize the chat options? =
Yes, you can set custom inquiry options from the plugin settings page.

= Does it work with any theme? =
Yes, it uses Tailwind CSS and Font Awesome for compatibility.

= Is tracking enabled? =
Basic analytics are included and can be enabled/disabled in settings.
